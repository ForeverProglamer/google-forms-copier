# google-forms-copier
App that copies answers from one google form to another

## Installation
Clone repository
```
git clone https://github.com/ForeverProglamer/google-forms-copier.git
```

Install dependecies
```
pip install -r requirements.txt
```

## Run app
```
python3 -m copier
```

## Run tests
Run all tests
```
python3 -m unittest
```

Run one test
```
python3 -m unittest copier.tests.test_source_page_scraper
```
